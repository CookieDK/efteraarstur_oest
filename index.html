Du kan gemme koden direkte som en .html-fil og åbne den i din browser. kan jeg ændre en reporitory til at være priat, således kun de med link kan se den? Kort svar: Nej, ikke på den måde. På platforme som GitHub, GitLab og Bitbucket findes der kun to primære tilstande for et repository: Public (Offentligt): Alle kan se og finde koden – også via søgemaskiner. Private (Privat): Ingen kan se koden, medmindre du eksplicit har tilføjet deres brugerkonto som samarbejdspartner (collaborator). Et direkte link giver ikke adgang til personer, der ikke er tilføjet. Hvordan du deler et privat repository Hvis du gør dit repository privat, er et link alene altså ikke nok. Modtageren skal have en konto på platformen, og du skal give vedkommende adgang: Gå til dit repository på platformen (f.eks. GitHub). Gå til Settings (Indstillinger). Rul ned til sektionen Danger Zone for at ændre synligheden til Private. Gå herefter til fanen Collaborators (eller Access Management). Tilføj personens brugernavn eller e-mailadresse. Derefter kan vedkommende tilgå projektet via linket, så længe vedkommende er logget ind. Alternativer, hvis du vil dele via link ("Unlisted") Hvis du har brug for en løsning, hvor alle med et unikt link kan se indholdet uden at skulle oprette en brugerkonto eller inviteres manuelt, kan du overveje følgende muligheder: GitHub Gist (Secret): Du kan oprette en "Secret Gist". Den er skjult over for søgemaskiner og oversigter, og alle, der har det direkte link, kan se koden. Det egner sig bedst til enkelte filer eller mindre kodestykker. Fildelingstjenester: Hvis du blot skal vise koden eller projektfilerne frem én eller få gange, kan du zippe projektet og dele det via Google Drive, Dropbox eller OneDrive med indstillingen "Alle med linket kan se". jeg har oprettet repository, hvordan er det, jeg får lavet html koden til en webside For at forvandle din HTML-kode til en rigtig webside via det repository, du har oprettet, skal du bruge en indbygget funktion på platforme som GitHub, der hedder GitHub Pages. Index.html (med stort I), forside.html eller tur.html.
